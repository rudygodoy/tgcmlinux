How to install Connection Manager

 

Installation Process

Some operations may require that you sign in as administrator (root) into the operating system.

Before to installation we suggest uninstalling any previous version of Connection Manager you had on your computer. Check that your laptop has connectivity before trying to install the version from the repository.

Note that Connection Manager needs to be installed, so not function properly if you are using a LiveCD distribution.
