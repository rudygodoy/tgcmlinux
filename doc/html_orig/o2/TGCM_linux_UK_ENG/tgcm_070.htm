Uninstall Connection Manager

 

Like the installation process, the uninstall process is simple. You can use any of the options given below:

  • To uninstall from Fedora Core

    • From GUI, go to Application >> Add /Uninstall Software. You'll see the Package Manager window and from there you will find the application and uninstall it.

    • From command line, execute next instruction:

yum remove connection-manager-uk

 

  • To uninstall from OpenSUSE:

    • From GUI, open YaST Tool, then go to Software tab, press on Install/ Uninstall Software. You'll see the Package Manager window and from there you will find the application and uninstall it.

    • From command line, execute next instruction

zypper remove connection-manager-uk

 

  • To uninstall from Ubuntu::
    • Go to System >> Administration >> Gestor de paquetes Synaptics. You'll see the Package Manager window and from there you will find the application and uninstall it.
    • From command line, execute next instructio:

apt-get remove connection-manager-uk

 

To uninstall properly you need to enter the operating system as root (administrator).
