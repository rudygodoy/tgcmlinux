Settings

 

«« BackNext »»


You can change different O2 Průvodce připojením options by clicking on Settings icon, located in the upper-left bar of the main window.

 

 

 

Figure xxx. Settings button.

 

 

Options

O2 Průvodce připojením

  • Start-up. By default, O2 Průvodce připojením does not launch whenever the session starts up. If you wish to activate this, just check the option Launch O2 Průvodce připojením when turn on the computer.

  • Connection. In this section, you also have the option of connecting automatically when you launch O2 Průvodce připojením by activating Connect when O2 Průvodce připojením is launched.

  • Updates. Select the option Automatically check for updates and news when I connect to activate the service of automatic notifications that can show information on application upgrades or news. Click on See updates and news button to view new updates. You can update the list manually using Check for recent updates and news option.

.


Networks and SIM card 

  • PIN Management

    • - Deactivate/Activate PIN: Here you can decide whether the program should request the PIN code for your SIM card or not, every time you connect. To do this you simply need to enter your current PIN.

      - Change PIN: To change the PIN code on your SIM card you simply enter the old PIN for your SIM card and then you also enter the new PIN that you want to set up.

  • Network technology selection: The option we recommend the most, which is set up by default, is Automatic, but if you want you can program O2 Průvodce připojením to only connect to the type of network you want clicking on Manual.
  • Operator selection: O2 Průvodce připojením automatically connects to the movistar network, but if you would like to select the operator you are going to connect with, clicking on Manual. After performing the search, you simply choose the operator you want from the list the program provides and press Select.

 

Display

You can check those services that will be ready in the menu area of your dock. Have in mind that the fisrt four will be shown, but the rest will be displayed under the menu displayed after clicking the More button.

You can sort the order of your services using the arrow buttons; besides you can also select a service with your mouse and drag it and drop to the place you want.

 

Devices

Here you can select the preferred GPRS/3G or Wi-Fi device that you want to use to connect to O2 Průvodce připojením. To do this, select the device that you want to connect with from the drop-down list.

 

My Internet connections

In this section you can create new GPRS/3G or Wi-Fi connections and change their parameters.

O2 Průvodce připojením includes one predefined GPRS/3G connection, called O2 Mobilní Internet. It also includes one predefined Wi-Fi connection, called O2 HotSpot. Remember that the predefined connection cannot be deleted or modified.

You can change the order of the list by selecting a connection and then clicking the up or down arrows, or by using the mouse to drag and drop the connection to the desired position. If you have a wired network, LAN connection will show on the list and as priority connection, unable to be removed or to be modified.

You can add new connections, edit, delete, import or export them. 

 

 How to add or change a GPRS/3G connection


In order to add or modify a connection, first you need to enter some configuration details for each connection. These are:

  • Identification: In this tab, you can set the connection name, username and password for network authentication. The Request password to connect option means the password is requested at connection, so it is not stored on O2 Průvodce připojením (if it is sensitive information, for example).

  • Connection details:  In this section you can specify the profile to use.

  • Network Configuration.

    • - Internet Access Proxy: The activated option by default is Do not use proxy in the connections. However, here you can configure the proxy parameters that you want.

    - Domain Names Server: If you choose the Server addresses assigned by the network option, it is assumed that the domain name service is automatically configured in the connection process. If you select the manual option, you have to enter the primary and secondary servers and the search domains.

If you do not know some of these details, you need to ask the network administrator.

If you access through a corporate Intranet or another type of network, contact your network administrator for this information.

 

Internet browsers that do not read the proxy configuration for the system must be manually configured according to manufacturer specifications.

 

How to export connections

Export connections allows you to generate a .tgcm file with the details on the connection selected. You can use this on another computer with O2 Průvodce připojením by applying the Import option.

To export a connection, access Settings >> My Internet connections, select a connection from the list and press the Export button. Then enter the name and place in which you want to save the file and press Save. This will generate a file that you can subsequently import at any time.

 The files that you can import and export by default for O2 Průvodce připojením have the .tgcm file extension.

 

How to import connections

Import connection allows you to add new connections to O2 Průvodce připojením from the files with .tgcm extension.

Once you have access Settings >> My Internet connections, press the Import button and a new window will open so that you can select the location of the file you want to import. This file will hold the information needed to configure the O2 Průvodce připojením connection. You can also import by directly double clicking on the file with a .tgcm file extension.

 

 

Services

In this section you can uninstall services available in O2 Průvodce připojením. To do that, click the checkbox near to the service icon. Remember that you can install the service again at any time using the same process. Changes made here by installing or uninstalling services affect to the dock-main window services zone.

 

Internet

You can configure Internet service and choose to open the web browser or not when you connect. You can also choose the connection to be used among the ones you have configured.

You can also configure the homepage by type in the text field provided for this option.

 

SMS

By activating the Use the following Short Message Center you can specify a different short message center from the default one. Also you can get a delivery notification activating Always request delivery notifications checkbox.

 

Favourites

You can add new Favourites, in addition to those that are already configured, by clicking on New and filling out the form.

In addition, you can modify, delete or import and export existing Favourites by clicking on one of these buttons to the right of the favourite screen.

 

How to export Favourites

Export Favourites creates a file with the extension .tgcm with the data necessary (URL address or directory of the application and the data of the associated connection) to access your local applications and favourite Internet addresses. You can use this file in another computer with O2 Průvodce připojením installed by using the Import option.

You can export each of the Favourites on the list by selecting the shortcut and clicking on Export. Then, enter the name of the file and the place you would like to save the file and click on Save.

 

How to import Favourites

Import Favourites can quickly add new Favourites to O2 Průvodce připojením with the use of files with the extension .tgcm.

To import a favorite, click on Settings >> Services >> Favourites >> Import and select the location of the file with the extension of .tgcm and click on OK. You can also import by directly opening a file with the extension .tgcm. If you do this, the file you just imported will be automatically added to the list of Favourites.

 

Firemní VPN

This service is not configured by default. To configure it, you need to contact your network administrator for the connection information.

 

Alerts

The Alerts section allows notifying you when you have reached the specific level of GPRS/3G traffic that you determined previously, for O2 network or roaming. By default, Alerts are enabled.

To set an Alert, select the volume of data from the dropdown list and then check the boxes to receive alerts when you reach the percentage of consumed data volume. Once established target range, click OK to be effective.

The alerts will be showed through a pop-up message (toaster).
