O2 Průvodce připojením Help

 

«« BackNext »»


The Help button informs you on how you can connect to the O2 Customer Services web site. You can also access the user manual for the O2 Průvodce připojením application by clicking on Help Topics.

This window also displays information about the Mobile Internet Device: manufacturer, firmware version and IMEI. You can also find information on Wi-Fi module built-in your computer.
