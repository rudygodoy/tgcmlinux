What is O2 Průvodce připojením?

 

Next »»



O2 Průvodce připojením is a free software that provides a smart manage of your Wi-Fi or 3G connections and gain mobile access to data services such as Internet, corporate Intranets or E-mail and O2 services with the same speed as a fixed workstation.

O2 provides mobile access at speeds equivalent to DSL through the use of UMTS/3G technology and the new HSPA/3.5G+/HSPA+ technology, while ensuring maximum coverage of the GPRS/EDGE network.

O2 Průvodce připojením is available in different versions and platforms, thus allowing you to use it with multiple terminals and operating systems. With this tool, you can connect from a number of devices, such as laptops or netbooks, using USB Modems, Embedded 3G Modems or with a telephone.

The aim of this guide is to summarize the basic concepts for installing, setting up and using O2 Průvodce připojením in its editions for Linux.
