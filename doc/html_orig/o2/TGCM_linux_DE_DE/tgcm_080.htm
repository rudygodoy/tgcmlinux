Hilfe den Mobile Connection Manager

 

«« ZurückNächster »»


Über die Schaltfläche Hilfe erhalten Sie Auskunft darüber, wie Sie die Internetseite des o2-Kundendienstes erreichen können. Sie können auf das Benutzerhandbuch für den Mobile Connection Manager auch zugreifen, indem Sie auf Hilfethemen klicken.

Von hier aus führt ein Link in diese Bedienungsanleitung. Außerdem gibt das Fenster Auskunft über die verwendete Version des Mobile Connection Managers, das verwendete Mobile Internetmodem, d.h. den Hersteller, die Firmware-Version und die IMEI. Sie finden weiter Information auf dem Wi-Fi-Modul Ihres Computers.
