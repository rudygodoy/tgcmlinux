Deinstallieren des Mobile Connection Managers

 

«« ZurückNächster »»


Zum Deinstallieren des Mobile Connection Managers können Sie jede der unten aufgelisteten Methoden benutzen:

  • Schritte für die Deinstallation der Anwendung in Fedora Core:
    • Klicken Sie im GUI auf Anwendung >> Software Hinzufügen/Entfernen. Daraufhin sehen Sie das Fenster Paket-Manager und von dort aus finden Sie die Anwendung, deinstallieren Sie diese.

    • Von der Befehlszeile aus führen Sie den nächsten Befehl aus:

yum remove tgcm -de

  • Schritte für die Deinstallation der Anwendung in OpenSUSE:
    • Klicken Sie im GUI auf das Werkzeug YaST, klicken Sie danach auf die Registrierkarte Software und anschließend auf Software Installieren/Deinstallieren. Suchen Sie im Fenster die Anwendung, Sie können diese deinstallieren.

    • Von der Befehlszeile aus führen Sie den nächsten Befehl aus:

zypper remove tgcm -de

  • Schritte für die Deinstallation der Anwendung in Ubuntu:
    • Klicken Sie im GUI auf Ubuntu Software Center klicken. In dem Fenster, auf installiert ist, wählen Andere Kategorie klicken und dann kannst du die Anwendung zu suchen und deinstallieren.

    • Von der Befehlszeile aus führen Sie den nächsten Befehl aus:

apt-get remove tgcm -de
