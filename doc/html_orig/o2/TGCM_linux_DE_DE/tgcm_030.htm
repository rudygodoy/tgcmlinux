Installation der Hardware

 

«« ZurückNächster »»


Bevor Sie neue Hardware installieren, versichern Sie sich, dass Sie die Mobile Connection Manager Software richtig installiert haben. Dieser Vorgang wird im Abschnitt Installation der Mobile Connection Manager

 

Voraussetzungen für die  Installation

  • einen Computer, mit einem USB 2.0 Anschluss für ein mobiles Internet-USB-Modem.
  • Internetfähiges Mobilfunkgerät: eine USB-Modem.
  • Die SIM-Karte muss für den o2 2G/3G/4G Dienst aktiviert sein

 

Internetfähigen Mobilfunkgerät

Das Bestriebssystem findet das Einschieben eines USB-Modems in Ihren Computer automatisch, sobald die Software installiert ist und jedes Mal, wenn Sie die Anwendung Mobile Connection Manager starten.

Führen Sie Folgendes aus, um die USB-Modem auf Ihrem Computer zu installieren:

  1. Schieben Sie zunächst den SIM-Chip von o2 in Ihr USB-Modem. 

Die Vorgehensweise beim Einschieben der USB-Modem und die Farben deren LED-Anzeige können je nach Modell unterschiedlich sein.

  1. Für das Einschieben von Mobilen Internetgeräten ist keine spezifische Konfiguration erforderlich. Schieben Sie die Modem in den USB-Port 2.0. Diese werden vom System und der Anwendung automatisch erkannt, sofern die Treiber für diese Geräte bei der Installation des Programms ebenfalls installiert wurden.
  1. Der Startvorgang beginnt nach dem Einschieben der USB-Modems. Die LED-Anzeige auf der Karte oder dem Modem beginnt zu leuchten um anzuzeigen, dass der Vorgang gestartet wurde (beachten Sie, dass die Farben der LED-Anzeige je nach Modell des USB-Modems unterschiedlich sein können.

  2. Sie müssen die SIM-Karte nach dem Start der Karte für das mobile Internet oder des USB-Modems im o2-Netz registrieren. Geben Sie hierzu Ihren PIN-Kode für Ihre SIM-Karte ein, wenn Sie vom System dazu aufgefordert werden.
