Wie startet man eine Internetverbindung mit dem Mobile Connection Manager

  1. Klicken Sie zum Öffnen der Anwendung Mobile Connection Manager auf Ihrem Computer auf das entsprechende Symbol.
  1. Wenn kein Gerät für das mobile Internet ausgewählt wird, können Sie es manuell durch Klicken auf Einstellungen >> Geräte auswählen.
  1. Klicken Sie auf Verbinden auf dem Dock und achten Sie darauf, dass die Empfangsanzeige grün leuchtet

Bei evtl. Problemen beachten Sie bitte den Abschnitt Fehlerdiagnose

 

Wie kann ich mit dem Mobile Connection Manager SMS versenden

  1. Klicken Sie zum Öffnen der Anwendung Mobile Connection Manager auf Ihrem Computer auf das entsprechende Symbol.
  1. Wenn kein Geräte für das mobile Internet ausgewählt wird, können es manuell durch klicken auf Einstellungen >> Gerät auswählen.
  1. Klicken Sie auf Verbinden auf dem Dock und achten Sie darauf, das die Empfangsanzeige grün leuchtet.
  1. Klicken Sie auf SMS im Bereich Dienste um die SMS-Nachrichten versenden bzw. verwalten zu können.

Bei evtl. Problemen beachten Sie bitte den Abschnitt Fehlerdiagnose
