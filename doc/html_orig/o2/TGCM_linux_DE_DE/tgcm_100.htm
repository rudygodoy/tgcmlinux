Der Mobile Connection Manager kann nicht gestartet werden

  • Der Gnome-Desktop muss installiert sein, damit die Anwendung Mobile Connection Manager korrekt funktionieren kann.

 

Die Komponente Mobile Manager ist nicht verfügbar

  • Es ist möglich, dass beim Ausführen der Anwendung Mobile Connection Manager eine Nachricht mit dem Hinweis angezeigt wird, dass die Komponente Mobile Manager nicht verfügbar ist. In diesem Fall müssen Sie den Computer neu starten oder den Daemon mit folgendem Befehl über die Befehlszeile starten: /etc/init.d/mobile-manager start.
  • Beachten Sie, dass Sie zum Ausführen dieser Aktion Root-Benutzer sein müssen. Sobald das Hintergrundprogramm gestartet ist, läuft die Anwendung korrekt.

 

Es kann keine Verbindung zum Internet hergestellt werden.

  • Falls Sie sich mit einem Mobilen Internet Modem verbinden möchten:

    • Vergewissern Sie sich, dass das Mobile Internetmodem korrekt an die USB 2.0 Buchse angeschlossen ist.

    • Ist das Gerät eingeschaltet? Schalten Sie es sonst bitte über den Startbildschirm ein.

    • Haben Sie die korrekte PIN für Ihre SIM-Karte eingegeben?

    • Haben Sie ausreichend Empfang?

  • Vergewissern Sie sich, dass Sie im Netzwerk registriert sind. Dies erkennen Sie im oberen, allgemeinen Bereich des Startbildschirms. Es sollte idealerweise eine 2G/3G/4G-Verbindung zum o2 Netz angezeigt werden. Falls Sie sich mit einem Mobiltelefon verbinden möchten:

    • Vergewissern Sie sich, dass die Verbindung zwischen dem  Computer und dem Telefon korrekt ist.

    • Haben Sie die korrekte PIN für Ihre SIM-Karte eingegeben?

    • Haben Sie ausreichend Empfang?

  • Vergewissern Sie sich, dass Sie im Netzwerk registriert sind. Dies erkennen Sie im oberen, allgemeinen Bereich des Startbildschirms. Es sollte idealerweise eine 2G/3G/4G-Verbindung zum o2 Netz angezeigt werden. Falls Sie mit einem Gerät verbinden möchten:

    • Vergewissern Sie sich, dass das Gerät korrekt gewählt worden ist. Informieren Sie sich dafür unter  Einstellungen >> Geräte >> 2G/3G/4G-Gerät.

Falls das Problem weiterhin besteht, nehmen Sie Verbindung mit Ihrem Kundenservice auf.Falls die Probleme weiter bestehen, setzen sie sich unter folgender URL-Adresse mit unserem Kundenservice in Verbindung: http://www.o2online.de/meta/hilfe/
