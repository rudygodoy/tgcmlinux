Mobile Connection Manager Installation

 

«« BackNext »»



Requirements for Installation

The minimum requirements for your computer to be compatible with the service Mobile Connection Manager for Linux are as follows:

  • Ubuntu 11.10

  • OpenSUSE 12.1

  • Fedora Core 16

In order to use Mobile Connection Manager you must be an o2 customer and have activated the GPRS/3G service. To find out this you can call o2 Customer Support service.

 

 

Installation Process

Some operations may require that you sign in as root (administrator) into the operating system.

Before to installation we suggest uninstalling any previous version of Mobile Connection Manager you had on your computer. Check that your laptop has connectivity before trying to install the version from the repository.

Note that Mobile Connection Manager needs to be installed, so not function properly if you are using a LiveCD distribution.
