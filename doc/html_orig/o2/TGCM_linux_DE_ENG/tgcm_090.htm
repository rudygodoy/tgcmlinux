How do I navigate through Internet using Mobile Connection Manager?

  1. Click on Mobile Connection Manager desktop icon to open the application.

  2. Connect your Mobile Internet Device to the computer.

  3. Now it will ask you for your PIN code from your SIM card (if it is activated); after you enter it, click on Accept. Wait for the network coverage indicator to turn green.

  4. Click on Available Networks button and select the 2G/3G/4G or Wi-Fi network that you want to connect to and click on Connect button.

  5. Open your Internet browser.

If you are having difficulties with this process, you can go to the Troubleshooting section for help.

 

How do I send a SMS using Mobile Connection Manager?

  1. Click on Mobile Connection Manager desktop icon to open the application.

  2. Connect your Mobile Internet Device to the computer.

  3. Now it will ask you for your PIN code from your SIM card (if it is activated); after you enter it, click on Accept. Wait for the network coverage indicator to turn green.

  4. Select SMS icon in the services zone and type in the details to send the message. If you have any questions refer to the SMS section of this document.

If you are having difficulties with this process, you can go to the Troubleshooting section for help.
