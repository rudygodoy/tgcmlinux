Connection Device Installation

 

«« BackNext »»



Before installing and configuring your Mobile Internet Device, make sure that you have installed the Mobile Connection Manager software correctly. This process is described in Mobile Connection Manager Installation section.

 

Requirements for Installation

  • PC or compatible.

  • Free USB 2.0 port for USB Mobile Internet Devices.
  • Mobile Internet Device.
  • SIM card activated in the o2 2G/3G/4G service.

 

 

Mobile Internet Device

Once the software has been installed, and whenever you start up Mobile Connection Manager, the operating system will automatically detect the insertion of a Mobile Internet Device in your computer.

Do the following to install the Mobile Internet Device in your computer:

  1. First, insert the o2 SIM card into your Mobile Internet Device.

The way to insert the SIM card in the Mobile Internet Device, and the color and the status of their LEDs, may vary according to the model in question.

  1. No specific configuration is needed before insertion for Internet Mobile Devices.  Simply insert the Modem and it will be automatically recognized by the system and the application, as long as the driver for this device was installed when you installed the program.
  1. The start-up process will begin after the Mobile Internet Device is inserted. The LEDs on the device will light up to indicate that the process is started (note that the LED color may vary depending on the model of the Mobile Internet Device).
  1. After starting up the Mobile Internet Device, you need to register the SIM card on the o2 network. To do this, enter your code PIN for your SIM card when the system prompts you to do so.

You only have three tries to correctly enter the code PIN for your SIM card, so take care when entering the PIN numbers.
