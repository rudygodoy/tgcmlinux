Troubleshooting

«« BackNext »»



I cannot start Mobile Connection Manager

  • For a properly run of the Mobile Connection Manager software you must have installed Gnome desktop.
|top|

 

Mobile Manager component is not available

  • It’s possible when you run Mobile Connection Manager application display a message saying the Mobile Manager component is unavailable. In this case you must restart the computer or start the daemon via the command line using the following command: /etc/init.d/mobilemanager start
  • Note that you must be root user to perform this action. Once up the daemon is launched, you can run the application properly.
|top|

 

I cannot start up a connection

  • Check that your Mobile Internet Device and computer are connected and that there is communication between them.
  • Check that you correctly set the configuration options you want to start service.
  • Check that the SIM card is correctly inserted into the Mobile Internet Device and it is inserted properly in the connection port of your computer.
  • If the problem persists, contact the Customer Support service at the following URL address http://www.o2online.de/meta/hilfe/
