Desinstalación de Escritorio Movistar

 

«« AtrásSiguiente »»



Para desinstalar Escritorio Movistar puedes utilizar cualquiera de los métodos que te mostramos a continuación:

  • Fedora Core

    • Desde el entorno gráfico, haz clic en Inicio, pulsa en Aplicaciones y a continuación en Agregar/Quitar Software. A continuación aparecerá la ventana del Administrador de Paquetes y desde ahí podrás localizar la aplicación y desinstalarla.

    • Desde la línea de comandos, debes ejecutar la siguiente instrucción:

yum remove escritorio-movistar-espana

  • OpenSUSE

    • Desde el entorno gráfico, abre la herramienta YaST, después en la pestaña Software pulsa en Instalar/Desintalar Software. En la ventana que aparece, localiza la aplicación y podrás desinstalarla.

    • Desde la línea de comandos, debes ejecutar la siguiente instrucción:

zypper remove escritorio-movistar-espana

  • Ubuntu

    • Desde el entorno gráfico, abre el Centro de software de Ubuntu y haz clic en el icono de Instalado y selecciona la categoría Otros. A continuación localiza la aplicación y desde ahí podrás y desinstalarla.

    • Desde la línea de comandos, debes ejecutar la siguiente instrucción:

apt-get remove escritorio-movistar-espana

 

Para desinstalar correctamente el software es necesario iniciar la sesión como root (administrador).
