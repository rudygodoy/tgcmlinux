Instalación de Escritorio Movistar

 

«« AtrásSiguiente »»



Requisitos de la instalación

Los requisitos mínimos con los que debe contar tu equipo para ser compatible con el software Escritorio Movistar para Linux son los siguientes:

  • Ubuntu 12.10

  • OpenSUSE 12.3

  • Fedora Core 18

Para poder utilizar el servicio Escritorio Movistar debes ser cliente de Movistar y tener activado el servicio GPRS/3G. Para comprobar este punto puedes llamar gratuitamente a Atención al Cliente de Movistar (1004).


Proceso de instalación

Es posible que algunas de las operaciones que realices requieran que te identifiques como root (administrador) dentro del sistema operativo.

Antes de proceder con la instalación, es recomendable que desinstales cualquier version previa de Escritorio Movistar que tuvieras instalada en tu equipo. Comprueba, además, que tu equipo tiene conectividad antes de agregar los repositorios.

Puedes consultar la información más reciente para obtener información actualizada acerca de cómo agregar repositorios e instalar Escritorio Movistar en tu distribución Linux en la nueva web de Ayuda de Escritorio Movistar para Linux.

Ten en cuenta que Escritorio Movistar requiere ser instalado. No funcionará correctamente si estás utilizando una distribución Linux LiveCD.
