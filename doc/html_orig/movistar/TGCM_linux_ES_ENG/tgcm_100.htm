Troubleshooting

 

«« BackNext »»



I can not start Escritorio Movistar

  • For a properly run of Escritorio Movistar software you must have installed Gnome desktop.

  • Sometimes certain user settings can make the application unstable. In these cases, you may not be able to start it. To solve this problem, try running from a Terminal tgcm-clean-config command to remove these settings. Run the tgcm-clean-config -h command to get help on this command and its optional parameters.
|top|

Mobile Manager component is not available

  • It’s possible when you run Escritorio Movistar application display a message saying the Mobile Manager component is unavailable. In this case you must restart the computer or start the daemon via the command line using the following command: /etc/init.d/mobile-manager start
  • Note that you must be root user to perform this action. Once up the daemon is launched, you can run the application properly.
|top|

I can not start up a connection

  • Make sure the Mobile Internet Device is properly inserted into the USB port. Make sure it is selected in Settings>> Devices.
  • Check that the SIM card is properly inserted in the Mobile Internet Device.

  • Check the Mobile Internet Device is registered on the network.

  • If you're connected via Wi-Fi, make sure you are typing the correct password and the connection that your device supports Wi-Fi (b / g / n).
  • If the problem persists, contact with the Help Online service. You can also call 1004 free of charge from a Movistar phone. For further information, please visit the following link where you can access to the new Escritorio Movistar for Linux Help website

 

|top|

I cannot check my Mobile Broadband number

  • In order to check your Mobile Broadband number is necessary for the SIM card is properly inserted in the Mobile Internet Device and it is connected to the computer.

  • Query of the Mobile Broadband number will not be possible if you have not installed the SMS service.

  • If you have contracted Internet Móvil Deberes rate you not be able to query the Mobile Broadband number as this rate does not include the sending / receiving SMS.

  • If you have a multiSIM card, the message with information about your Mobile Broadband number will be received by the main SIM card.
