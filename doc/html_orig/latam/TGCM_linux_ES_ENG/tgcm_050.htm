Escritorio Movistar Latam Services

 

«« BackNext »»



Depending on the services you selected in the installation process, these will be available in the service area of Escritorio Movistar Latam main window.

The different services and settings of Escritorio Movistar Latam are available depending on the place where you are accessing them, not all are available in all locations.

 

SMS

SMS allows to send text messages to different contacts you have saved in the Escritorio Movistar Latam Contacts service and store all sent and received messages.

Escritorio Movistar Latam supports sending and receiving linked messages, which lets you write or receive messages much longer connecting up to six text messages.

SMS application is distributed over five tabs: New, Inbox, Sent, Conversations and Draft.

 

New Message


In the New tab have available a view of the Contacts service, a To: box where will the contacts recorded selections and a Message: box where you can type the text to send.

You can add contacts to the list of the To: box in different ways:

  • Double clicking on the contact you want.
  • Through the context menu by selecting one or more contacts and then pressing the right mouse button.
  • Manually, entering phone numbers separated by commas from the box To:

When writing a message, you can see how they count the characters in the Message: box to show how many have entered and the number of messages to be sent eventually.

Messages are saved as a draft by clicking the Save button. If you want to delete the message you were writing, press the Clear All button (erase both the message and the recipients selected).

To send the message you've write, click the Send button. The message is automatically saved in the Sent folder.

To receive a notification when the message is delivered to the recipient, you can enable Delivery notification box.

 

 Inbox


Inbox tab shows you your inbox with a list of all the messages sorted in reverse chronological order, ie see the latest always the first. Note that if you click on any column, you can reorder in ascending / down. If you have any pending message reading, you will be shown in bold.

In the same tab, next to the Inbox, you may see a number, which corresponds to the number of messages you have unread.

Below the list of messages received is a panel will show the details of the message that you selected. The detail shows the body of the message sender data and time of receipt. If the number belongs to any contact you have in the Contacts service, displays your name instead of number.

Clicking the secondary mouse button on the body of the message will enable the Use details option. Tick the message detail of your interest and then select the action what you want (depending of its content): Send Message, Add to contacts list, Assign to contact, Open URL, Add to Favourites or Send an e-mail.

If you want to Reply to the message, opens the New tab with the number of the sender in the To: box.

You can also delete messages for free using the Delete selected button, if you want to delete multiple messages, use the mouse and <Shift> and <Ctrl> keys to select messages to delete.

 

Sent

Sent tab will show you a list of sent messages in reverse chronological order (ie always see the most recent first). You can select messages to read its contents in the box below for details.

In the same tab, along with envoys may be given a number, which corresponds to the number of messages waiting to be sent to you.

When you select a message, the buttons Edit and Delete selected. In the first case, clicking on Edit will open the New tab with the content of the message sent and To: box will the original recipients. In the second case, clicking on Delete selected will delete selected messages from the list of envoys. If you want to delete multiple messages, use the mouse and <Shift> and <Ctrl> keys to select messages to delete.

 

Conversations


Conversations tab you will find the messages you exchanged with your contacts, grouped together so that you can keep better track of them chronological.

In Contacts panel show you sorted alphabetically, contacts with whom you have exchanged messages. If you have contact available on the Contacts service, the application will show the name contact instead of your phone number. Next to contact you will also see a number, indicating the number of messages exchanged with that contact.

In Messages panel you will see the messages along to the recipient name or number and date of receipt or dispatch of the message. You can select whether to display the messages sent, received, all or none checking or unchecking the boxes in your Inbox: and Sent: By default, you will show all messages in chronological order.

To make easy interpretation of the talks, Escritorio Movistar Latam shows you the messages in different colors, depending on whether the received messages (colored with black text) or mailed (dark colors with white text).

You can select one or more contacts, using the mouse and <Ctrl> and <Shift> keys and thus it will show the messages exchanged with all the selected contacts.

When you select a message will activate the Delete button and select Reply. In the first case, clicking on Delete selected you can delete the message or messages you've selected the Messages panel. In the second case, pressing the Reply button will open the new tab with the number or contact name of the sender in the To: box.

 

 Drafts

In the Drafts tab will show you a list of saved messages.

When you select a message in this folder will display its contents in detail and the panel will activate the Edit button. When editing a message from this folder will open the New tab with the contents of the To: and Message: boxes  the message in draft.

In the same tab, along with draft may be given a number, which corresponds to the number of messages you have stored in this folder.


Contacts

Contacts service allows you to save and see information on your contacts on your computer, whether you import them from other sources and contacts who have registered themselves on Escritorio Movistar Latam application.

By default, you have defined fields for each of the contacts are: name, number and email address.

From the button bar can create a new contact, you can modify with the Modify button, and pressing Delete you can delete as you wish. You can also send an email or send an SMS to the contact that you have selected. These options are available through contextual menus too.

You can import contacts from the Internet Mobile Device SIM card itself. If during the import Escritorio Movistar Latam detects duplicate contacts are always prevail have already registered in Contacts.

In the top right of the window you have the Search tool to find a quick and easy any contact, because the search results are being filtered in the contact list, showing the match progress in all fields ( the coincidence is established from the first character you type).

Note that if you double-click on a contact window will open so you can edit the data. Also, to delete or send an email to multiple contacts, you can use the mouse in combination with <Shift> and <Ctrl> keys.

For the “telephone” field, the preference in import order from E-mail applications will be: 1) mobile, 2) office, 3) home.

 

 

Favourites

Favourites is the quick way of finding and using your Internet address or favorite applications in Escritorio Movistar Latam. Clicking on Favourites you will enter to the service window where you can see all accesses configured by default. Here you can add new favourites and edit its settings.

 

 

Other Services

By clicking on More button from dock-main window, the rest of services (installed but not available in the main window) are displayed.

The different services and settings of Escritorio Movistar Latam are available depending on the place where you are accessing them, not all are available in all locations.
