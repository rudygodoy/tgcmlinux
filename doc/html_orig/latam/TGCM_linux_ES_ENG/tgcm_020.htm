Escritorio Movistar Latam Installation

 

«« BackNext »»



Requirements for Installation

The minimum requirements for your computer to be compatible with the service Escritorio Movistar Latam for Linux are as follows:

  • Ubuntu 12.10

  • OpenSUSE 12.3

  • Fedora Core 18

In order to use Escritorio Movistar Latam you must be a Movistar customer and have activated the GPRS/3G service. To find out if you are activated in this service, you can call the Movistar Customer Hotline of your country.

 

Installation Process

Some operations may require that you sign in as root (administrator) into the operating system.

Before to installation we suggest uninstalling any previous version of Escritorio Movistar Latam you had on your computer. Check that your laptop has connectivity before trying to install the version from the repository.

You can visit the new Escritorio Movistar for Linux Help website to know more details about the installation process and each supported distribution, there will also find additional updated information about Escritorio Movistar Latam.

Note that Escritorio needs to be installed, so not function properly if you are using a Linux LiveCD distribution.
