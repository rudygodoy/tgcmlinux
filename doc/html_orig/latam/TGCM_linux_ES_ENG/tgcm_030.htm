Connection Device Installation

 

«« BackNext »»



Before installing and configuring your Mobile Internet Device, make sure that you have installed the Escritorio Movistar Latam software correctly. This process is described in Escritorio Movistar Latam Installation section.

 

Requirements for Installation

To install the application, you need the following:

  • Computer compatible with a USB 2.0 port free.
  • Mobile Internet Device.
  • SIM card activated in the Movistar GPRS/3G service.

 

Mobile Internet Device

Once the software has been installed, and whenever you start up Escritorio Movistar Latam, the operating system will automatically detect the insertion of a Mobile Internet Device in your computer.

  1. First, insert the Movistar SIM card into your Mobile Internet Device.

Figure 1. Example of how to install the SIM card in a Mobile Internet Device

The way to insert the SIM card in the Mobile Internet Device, and the color and the status of their LEDs, may vary according to the model in question.

  1. No specific configuration is needed before insertion for Internet Mobile Devices.  Simply insert the device into the USB 2.0 port, and it will be automatically recognized by the system and the application, as long as the drivers for that device was installed when you installed the program.
  1. The start-up process will begin after the Mobile Internet Device is inserted. The LEDs on the device will light up to indicate that the process is started (note that the LED color may vary depending on the model of the Mobile Internet Device).
  1. After starting up the Mobile Internet Device, you need to register the SIM card on the movistar network. To do this, enter your code PIN for your SIM card when the system prompts you to do so.

To enter the PIN code you have only 3 tries, so be careful when entering these numbers. Used up all 3 attempts, the SIM card will be locked and you will must enter the PUK code to unlock it. If you enter your PUK code incorrectly SIM card becomes unusable and must request for a duplicate thereof.
