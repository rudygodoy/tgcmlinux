Using Escritorio Movistar Latam

 

«« BackNext »»



 

How to start up Escritorio Movistar Latam

You can start up Escritorio Movistar Latam in two ways:

  • From the GUI, you have a shortcut to the application. Click on Applications menu, then click on Internet >> Escritorio Movistar Latam.

  • If you use Ubuntu or Fedora (Unity shell), click on Dash home or Activities, type escritorio movistar, and then click on Escritorio Movistar Latam icon.

Figure 2. Escritorio Movistar Latam application icon

  • From the command line, open a Terminal and type in the command line tcgm –cc ( “cc” is country code) and then press Enter.

Escritorio Movistar Latam also includes multi-language support (spanish and english). To change the language, from the task bar, go to System >> Administration >> Language Support. Close Escritorio Movistar Latam and reopen it to check that the language has changed. Note that the language change has a global effect on the whole system, which is now displayed in the language you have chosen.

It is necessary for your Linux distribution has the Gnome desktop installed for the proper works  of the Escritorio Movistar Latam application.

For statistical purposes, Movistar processes anonymous user details related to the use of Escritorio Movistar services, connection details, operating system and software versions, for the sole aim of identifying possible improvements in the services and enhancing user-friendliness. No personal details are processed under any circumstances.

 

Composition of the graphic interface

Escritorio Movistar Latam main window is composed of three different zones:

Figure 3. Escritorio Movistar Latam main window

 

  • General Zone. This zone shows the device used, traffic data meter with the used and available data and Settings and Help buttons.

  • Services Zone: Here you can access all the available services and features offered by Escritorio Movistar Latam.

  • Connectivity Zone: Here you can connect and check information on the active connection.

A series of popup messages will appear whenever something happens; for example, whether you receive a SMS, whether the preset WWAN traffic level has been reached or whether there has been a problem establishing the connection or it has ended unexpectedly.

 

General Zone

Figure 4. General zone detail


General zone is placed on top of main window of Escritorio Movistar Latam; from here you can to access to general options. Into General zone, there are distinct zones: Device zone, Traffic data meter and Settings and Help buttons.

  • Device zone: in this place, you can see a button with the device brand, an icon that shows the GPRS/EDGE/3G/3,5G/3,5G+ or HSPA+ coverage level and the operator. If you are in roaming mode, you can see an icon triangle.

By clicking on the button with the device brand, a contextual menu shall appear. This menu shall offer you the following options:

- Deactivate/Activate PIN: Here you can decide whether the program should request the PIN code for your SIM card or not, every time you connect. To do this you simply need to enter your current PIN.

- Change PIN: To change the PIN code on your SIM card you simply enter the old PIN for your SIM card and then you also enter the new PIN that you want to set up.

- Network technology selection: The option we recommend the most, which is set up by default, is Automatic, but you can set up Escritorio Movistar Latam to only connect to the type of network you want.

- Carrier selection: Escritorio Movistar Latam automatically connects to the Movistar network, but you can select the carrier you are going to connect with, press Manual. After performing the search, you simply choose the carrier you want from the list the program provides and press Select.

- Turn on/off the device.

  • Traffic data meter: In this zone you can see your available and total data purchased and data used. Check Traffic section for further information.
  • Settings: Here you can configure all the options of Escritorio Movistar Latam application. See Settings  section for further information.
  • Help: Here you can see what version of the application you are using, and get help on how to use Escritorio Movistar Latam correctly. See the Help section in this document.

 

Services Zone

Service zone is located on a horizontal strip to the right and contains icons that give access to the services provided by Escritorio Movistar Latam.

Figure 5. Services zone detail

  • SMS: You can send and receive SMS from your computer with the SMS service. This service is available only if you connect through a Mobile Internet Device.
  • Contacts: This service allows you to save and view information about your contacts, whether they can import from other sources as well as contacts that have been registered with Escritorio Movistar.
  • Favourites: Favourites are a quick way of finding and using your Internet address or favorite applications in Escritorio Movistar Latam.

Clicking on More button you can access to additional services offered by Escritorio Movistar Latam.

The different services and settings of Escritorio Movistar Latam are available depending on the place where you are accessing them, not all are available in all locations.

 

 

Conectivity Zone

Figure 6. Connectivity zone detail


In this zone the application window shows tags with general information on the connection you are using. This information is updated automatically. The following data is shown:

  • Connection. Once connected, shows the name of the active connection you are now using to connect to Escritorio Movistar Latam, its status (connected o disconnected), and the strength of the signal and its speed.
  • Available networks. By clicking on this button, a list of the available networks shall be shown. The signal quality is represented by an icon to the left. If it's a Wi-Fi, to the right is an icon indicating their level of safety (secure network or unprotected network). Select the network you want to connect and then click on Connect button.
  • Connect. By clicking on Connect button you can establish a connection through the smart connector manager, preferably in the order set out therein. Check My Internet connections further information.

Escritorio Movistar Latam lets you connect to through different types of connections.

  • WWAN: Allows you connect to different Movistar network technologies (GPRS/EDGE/3G/3.5G/3.5G+ or HSPA+).
  • Wi-Fi: Allows you to make a wireless connection to any of the Wi-Fi networks within reach at the time. You can use Mobile Internet Devices with this functionality built-in or the Wi-Fi technology on your computer.
  • LAN: Detects if you have an IP address assigned and it is used to access to the services of Escritorio Movistar. If the application finds that a LAN connection is available, this will let you to connect through a LAN wherever possible.

For more information on how Create or Eliminate connections or change their order of preference, see My Internet connections section.


Popup Messages

When you have a Mobile Internet Device selected as connecting device, and is properly registered on the network, Escritorio Movistar Latam will display a popups at the bottom reporting you of certain events, such as:

  • Incoming SMS
  • Carrier messages (e.g. billing period, credit expired).
  • If a problem is detected when the connection is lost or if it unexpectedly.
  • If you have established traffic alerts, when you reach the limit of data transfer volume set.

In some cases, these messages you receive will be clickable and doing click on them with the mouse will open an Escritorio Movistar Latam service or directly opens the Internet browser.



Keyboard shortcuts

Below we detail keyboard shortcut defined for Escritorio Movistar software, to improve accessibility for handicapped people.

  • Connect  -> CTRL+C
  • Disconnect  -> CTRL+D
  • Exit Escritorio Movistar -> CTRL+Q

These keyboard shortcuts are functional when the Escritorio Movistar main window is selected.
