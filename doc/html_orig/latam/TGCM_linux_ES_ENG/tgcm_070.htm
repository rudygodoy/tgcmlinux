Escritorio Movistar Latam Uninstallation

 

«« BackNext »»



In order to uninstall Escritorio Movistar Latam you can follow these steps:

  • Fedora Core:

    • From GUI, go to Application >> Add /Uninstall Software. You'll see the Package Manager window and from there you will find the application and uninstall it.

    • From command line, execute next instruction:

yum remove tcgm –cc ( “cc” is country code)

 

  • OpenSUSE:

    • From GUI, open YaST Tool, then go to Software tab, press on Install/ Uninstall Software. You'll see the Package Manager window and from there you will find the application and uninstall it.

    • From command line, execute next instruction:

zypper remove tcgm –cc ( “cc” is country code)

 

  • Ubuntu:
    • From the GUI, click on Ubuntu Software Centre. In the window, click on Installed, select Others category and then you can locate the application and uninstall it.
    • From command line, execute next instruction:

apt-get removetcgm –cc ( “cc” is country code)

 

In order to uninstall properly Escritorio Movistar Latam software it is necessary log in as user root.
