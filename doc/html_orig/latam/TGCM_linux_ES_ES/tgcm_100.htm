No puedo iniciar Escritorio Movistar Latam

  • Para un correcto funcionamiento de Escritorio Movistar Latam es necesario tener instalado el escritorio Gnome.

  • En ocasiones, ciertas configuraciones de usuario pueden hacer inestable la aplicación. En estos casos, es probable que no puedas iniciarla. Para solucionar este problema, prueba a ejecutar desde un Terminal el comando tgcm-clean-config para estas eliminar configuraciones. Ejecuta tgcm-clean-config -h para obtener ayuda acerca de este comando y sus parámetros opcionales.

El componente Mobile Manager no se encuentra disponible

  • Es posible que al ejecutar Escritorio Movistar Latam se muestre un mensaje relativo a que el componente Mobile Manager no se encuentra disponible. En este caso deberás reiniciar el equipo o bien arrancar el demonio a través de la línea de comandos utilizando la siguiente instrucción: /etc/init.d/mobilemanager start
  • Observa que es necesario ser usuario root para poder llevar a cabo esta acción. Una vez levantado el demonio ya podrás ejecutar la aplicación de forma correcta.

No puedo conectarme a Internet

  • Comprueba que el Dispositivo Internet Móvil está correctamente insertado en el puerto USB. Asegúrate de que está seleccionado en Configuración >> Dispositivos.
  • Comprueba que la tarjeta SIM está correctamente insertada en el Dispositivo Internet Móvil.

  • Comprueba que el Dispositivo Internet Móvil está registrado en la red.

  • Si estás conectando por Wi-Fi, asegúrate de que estás escribiendo correctamente la contraseña de la conexión y que tu dispositivo W-Fi es compatible (b/g/n).

  • Si el problema persiste, contacta con el servicio de Atención al Cliente de Movistar de tu país. También puedes obtener más ayuda visitando la web de Soporte de Escritorio Movistar para Linux.
