Cómo navegar por Internet utilizando Escritorio movistar

  1. Abre la aplicación de Escritorio movistar en tu ordenador, pulsando sobre el icono correspondiente.

  2. Selecciona tu Dispositivo Internet Móvil pulsando en Configuración >> Mis dispositivos. A continuación te pedirá el código PIN de tu tarjeta SIM, si éste se encuentra activado; una vez introducido, pulsa Aceptar y espera a que el indicador de cobertura de la red del Visor de Información se ponga en verde.
  1. Por último pulsa el botón movistar Internet de la zona de servicios y se abrirá el navegador web de tu equipo con la página de inicio del mismo.

Puedes consultar el apartado Resolución de problemas si has encontrado alguna dificultad en este proceso.

Cómo enviar un correo electrónico utilizando Escritorio movistar

  1. Abre la aplicación de Escritorio movistar en tu ordenador, pulsando sobre el icono correspondiente.

  2. Si no hay dispositivo de conexión seleccionado, puedes seleccionarlo pulsando en Configuración >> Mis dispositivos. Consulta el punto Cómo navegar por Internet utilizando Escritorio movistar para saber cómo seleccionar correctamente los dispositivos de conexión.

  3. A continuación, pulsa en el botón Correo Móvil de la zona de servicios. Si es la primera vez que accedes al servicio tendrás que introducir Teléfono y Clave.

Si todavía no eres usuario de los servicios movistar o has olvidado tu clave de acceso, solicita tu clave enviando con tu móvil un mensaje de texto gratuito al 770 cuyo texto sea la clave que desees. Ejemplo, envía Azules al 770 para que sea tu clave de acceso.

Puedes consultar el apartado Resolución de problemas si has encontrado alguna dificultad en este proceso.

Cómo enviar un mensaje de texto o mensaje multimedia utilizando Escritorio movistar

  1. Abre la aplicación de Escritorio movistar en tu dispositivo, pulsando sobre el icono correspondiente.

  2. Si no hay dispositivo de conexión seleccionado, puedes seleccionarlo pulsando en Configuración >> Mis dispositivos. Consulta el punto Cómo navegar por Internet utilizando Escritorio movistar para saber cómo seleccionar correctamente los dispositivos de conexión.

  3. Pulsa el icono Mensajes de la zona de servicios y rellena los datos para enviar el mensaje. Si tienes alguna duda, consulta el apartado Mensajes de este documento.

Puedes consultar el apartado Resolución de problemas si has encontrado alguna dificultad en este proceso.

Cómo enviar un mensaje de texto vía web utilizando Escritorio movistar

  1. Abre la aplicación de Escritorio movistar, pulsando sobre el icono correspondiente.
  1. Selecciona tu Dispositivo Internet Móvil pulsando en Configuración >> Mis dispositivos. A continuación te pedirá el código PIN de tu tarjeta SIM, si éste se encuentra activado; una vez introducido, pulsa Aceptar y espera a que el indicador de cobertura de la red del Visor de Información se ponga en verde.
  1. Accede al Acceso directo Mensajería Web. Si es la primera vez que accedes al servicio tendrás que introducir Teléfono y Clave.
  1. Una vez que hayas accedido a la página web selecciona el botón Acceder.

Si todavía no eres usuario de los servicios movistar o has olvidado tu clave de acceso, solicita tu clave enviando con tu móvil un mensaje de texto gratuito al 770 cuyo texto sea la clave que desees. Ejemplo, envía Azules al 770 para que ésa sea tu clave de acceso.

Puedes consultar el apartado Resolución de problemas si has encontrado alguna dificultad en este proceso. Para configurar el Acceso directo Mensajería Web puedes consultar el apartado Datos de Usuario.
